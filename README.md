Portal del Proveedor — Demo navegable para ICB Food Service.

Stack: Next.js 16 + React + TypeScript + Tailwind 4 + Lucide React.

Data hardcodeada, cero backend, 100% visual e interactivo.

## Setup

```bash
npm install
npm run dev
```
